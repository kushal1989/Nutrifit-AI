# 🥗 NutriFit AI

NutriFit AI is an AI-powered Nutrition & Fitness Guidance Assistant developed using **Python, Streamlit, LangChain, and Google Gemini AI**.  
The application provides intelligent and interactive health-related guidance through a conversational chatbot interface.

This project focuses on combining **Generative AI** with health and wellness applications to help users get personalized suggestions related to fitness, nutrition, healthy diets, calorie awareness, and workout recommendations.

---

# 🚀 Features

- 🥗 Nutrition & Diet Guidance
- 🔥 Calorie Information Support
- 💪 Weight Loss & Muscle Gain Suggestions
- 🏋️ Workout Recommendations
